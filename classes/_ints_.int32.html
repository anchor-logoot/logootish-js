Class Int32

An IntType that restricts the number to 32 bits by using an Int32Array.

inheritdoc
example
const a = new Int32(5)
console.log(a.toString()) // 5
a.add(10).sub(8)
console.log(a.toString()) // 7
const b = new Int32(3)
console.log(a.cmp(b)) // 1
